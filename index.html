Selected Works.

VoiceMap
(Civic GeoTech-Portal & Authority Dashboard)

Mapbox • React/Next.js • Spring Boot • MongoDB • GeoJSON • Pincode Search • Authority Mapping

VoiceMap is an interactive civic mapping platform combining spatial datasets (states, districts, constituencies, pincodes) with representative contact information. Users can find their constituency by pincode, district, or name and view their MP/CM details.

What I built
  • Fast client-side rendering using Mapbox GL + static GeoJSON.
  • Pincode-first search with centroid jumping & prefix suggestions.
  • Authority mapping pipeline (OCR + fuzzy match + data cleaning).
  • Optimized geo imports & 2dsphere index queries.
My Role

Design + Full-stack development (data ingestion, backend APIs, frontend map UX, search logic).

Challenges & Solutions
  • Performance vs accuracy → static geo tiles + DB only for lookups.
  • Messy authority data → custom cleaning + validation workflow.
  • Pincode UX → fast numeric auto-detection for accuracy.
